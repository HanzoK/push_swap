# push_swap

## Project Overview
Sort a list of integers using **two stacks** and a limited instruction set, while minimizing the number of operations via QuickSort algorithm.

## Concepts Covered
- Sorting algorithms
- Complexity analysis
- Stack operations
- Optimization strategies

## Build
```
make
```
## Usage

```
./push_swap 4 67 3 1
```

## What I learned

- Algorithmic thinking under constraints
- Tradeoffs between simplicity and performance
